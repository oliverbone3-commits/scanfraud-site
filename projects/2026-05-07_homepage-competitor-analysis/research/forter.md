## Patterns to learn from
- One single global CTA ("LET'S TALK") repeated 4-5 times — extreme funnel discipline.
- Pair every customer quote with hard numeric outcomes inside the same card — prose + metrics, never just prose.
- Frame customer base as a network (Wayfair, Instacart, Adidas "fuel it") instead of a logo wall — turns clients into proof of network effect.
- Mid-page sitemap block (Block 12) groups all product pages under suite labels — doubles as in-page nav for users who skim past hero.
- "LIVE DATA" widget (Block 16) — using a public benchmark dashboard as a content/lead asset rather than a static stat.

## Patterns to skip
- Animated number counters that render as "0" in DOM until scrolled — bad for scrapers, screen readers, and SEO snippets.
- No logo wall — for a less-famous vendor (ScanFraud), explicit logos still pull weight that name-drops in prose can't.
- Hero offers no secondary action — for a smaller brand, missing a "see how it works" / docs path forces every visitor to a sales call.
- Heavy reliance on lifestyle photography (model in sweater) over product UI — looks luxe but tells you little about the product in 2 seconds.
- "Built for what's next" / "Smarter operations" headings are too generic — almost interchangeable with any AI vendor.
