# Design Tokens Reference

Quick reference for CSS variables used across the site. Source of truth: `shared.css`.

## Layout

| Token | Value | Usage |
|-------|-------|-------|
| `--col-width` | 1160px | Main container width |
| `--side-pad` | 140px | Horizontal page padding |
| `--cell-pad` | 40px | Internal cell/section padding |
| `--nav-h` | 65px | Fixed nav height |

## Typography

| Token | Value | Usage |
|-------|-------|-------|
| `--font-g` | 'Geist' | Headings, titles, descriptions |
| `--font-i` | 'Inter' | Body text, eyebrows, navigation |
