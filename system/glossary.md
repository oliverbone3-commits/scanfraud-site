## Domain Terms

| Term | Definition | Context |
|------|-----------|---------|
| Chargeback | Reversal of a card payment initiated by the cardholder | Dispute Resolution context |
| False decline | Legitimate transaction blocked by fraud rules | Transaction Monitoring context |
| Cascading | Re-routing a failed transaction to an alternative provider | Routing context |
| PSP | Payment Service Provider | General |
| 3DS / 3D Secure | Cardholder authentication protocol | Compliance context |
| PSD2 / SCA | EU payment regulation / Strong Customer Authentication | Compliance context |
| PCI DSS | Payment Card Industry Data Security Standard | Security context |
| BIN | Bank Identification Number (first 6-8 digits of card) | BIN Checker tool |
| KYC | Know Your Customer verification | Compliance context |
| AML | Anti-Money Laundering | Compliance context |
